Woocommerce - Bulk Price Updater
Version : 1.0

/* Important Notes  */

Plugin only works for woocommerce Bulk Price updation.

How to use -
1. First you need to install woocommerce plugin in your wordpress.
2. Now install bulk-price updater plugin.
3. Activate and it will appear in side bar (For reference see screenshot).
4. Enjoy!!
